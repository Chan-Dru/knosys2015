"Behind every code is an enigma".

Do puzzles drive you crazy? Are you a coding geek? Wanna try something new? If the answer is yes , then of course you belong here.

Here is an event "Enigma" awaiting you to hone your skills. So get ready to unravel the enigma.

  • Time duration of the event : 1 hr
  • A team consists of only one person.
  • Participants can code in either C or C++ .
  • Selection of candidates is based on number of questions answered and their weightage. In case of tie, execution time would be considered.

The event will be conducted in one phase.

  • The participant will be given 3 questions along with their associated weightage and 9 unordered modules of code.
  • Participants have to choose 3 modules for each question, join to form the code to some extent & rest should be added by the participant to complete the code for each question.
  • All the modules of code should be used and each module can be used only once.
