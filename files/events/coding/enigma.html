  • Participants must pre-register online for the event.
  • Individual participation is only allowed.
  • Participants can code in either C or C++ language.

The event will be conducted on-site.

  • The objective is to form 3 programs from the given code snippets.
  • 3 questions and 9 code snippets will be provided.
  • The participant has to use the provided snippets and build the programs corresponding to each question.
  • 3 snippets have to be used per question and each snippet can be used only once.
  • These snippets provide only the essential logic behind the problem statements. The rest of the coding should be done by the participant.
