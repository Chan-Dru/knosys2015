Win Adroit

  • Introduction
  • Rules&Regulation
  • Format

Do you think you know all there is to know about Windows? Come evaluate yourself here at Win Adroit where you are put through myriad windows tasks. See how well you think you are!!

  • Each participant will be provided a computer terminal with windows operating system
  • Internet connection will provided to each terminal
  • Each participant will be given a theme which will be pertaining to windows operating system.
  • A time limit will be given within which the participant must search the theme in the internet and get to know about the theme.
  • A set of tasks to be performed by the participant will be issued after the given preparation time.
  • The participants must complete the tasks within the stipulated time.
  • The participant who completes maximum number of tasks would be declared winner.

This event includes:

  • Computer Troubleshooting.
  • Windows Customization.
  • Testing Windows handling skills of participants.
  • Testing basic knowledge on command prompt and batch file programming.
  • Time duration 1 hour.
  • Individual or Team participation (2 members).
  • Total number of levels : 3
  • Level 1 (Easy): Each carry 1 mark.
  • Level 2 (Moderate): Each carries 2 mark.
  • Level 3 (Hard): Each carries 3 mark.
  • OS: Windows Platform (Windows 7).
  • The judges reserve right to disqualify if any discrepancies are found.
