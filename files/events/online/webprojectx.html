To all the aspiring designers out there, here is a chance to unravel that creative genius in you. We give you the set-up of a website, you build it. Good is not enough! It has to remarkable, because so is the prize!

  • Code plagiarism is strictly discouraged at all instances.
  • Usage of libraries other than the specified libraries present in RUBRIC will lead to disqualification of the entry.
  • Usage of pre-built templates for frontend design will lead to dis-qualification of the entry.
  • Entries containing irrelevant images or other media files will not be taken for consideration.
  • Usage of Frameworks (such as Laravel, Ruby on Rails, Play, Zend, etc.) for development is encouraged, but the frameworks used must be clearly specified on the Readme File.
  • All entries must contain a Readme file which contains vital information regarding the Languages used, Database used and libraries included.
  • Source Folder size should not exceed 150MB.

    • RUBRIC

  • No language or framework constraints.
  • Participants can host their entries in any of the following sites or on sites of their convenience.
    • (000webhost.com, 5gbfree.com, openshift.com, amazon-aws)
      The following libraries can be used for Development
    • Twitter Bootstrap
    • JQuery
    • Font Awesome (Icons & Fonts)
    • Node.js
    • All web-supported Fonts
  • The entries are required to be Firefox 30+ compatible.
  • Entries should be live and available in the link provided at least till 3rd February 2015.
  • Clearly specify the team name and participant names in the Readme file.
  • Participants are required to enroll themselves at knosys.in to participate.
  • A team of maximum two members can participate.
  • A single problem for which the participants need to build a web application will be provided on enrollment.
  • Submissions will be entertained on or before 25th January 2015.
  • Participants are expected to send the link to the project, source code folder along with a readme file within the stipulated time.
  • A maximum of two entries is permitted for a team.
